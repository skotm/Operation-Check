## プロジェクト概要

`wis viewer` は、気象庁の公開データをもとに、全国の気象情報を地図上で見やすく可視化するWebアプリケーションです。  
気温、降水量、風向・風速、台風情報、注意報・警報、雨雲レーダー、全国ランキングなどを、ひとつの画面で直感的に確認できます。

HTML / CSS / JavaScript だけで動作するため、バックエンドサーバーを用意せずに利用できます。

## 詳細機能 (Features)

### 1. 気温・降水量の可視化
- **気温レイヤー**: 全国の観測点を温度に応じた色分けで表示。
- **降水量レイヤー**: 観測値モードと雨雲レーダーモードを切り替え可能。
- **詳細カード表示**: 各観測点をクリックして、観測値の詳細を確認可能。

### 2. 風向・風速の可視化
- **風向・風速レイヤー**: 観測点ごとの風速をアイコン付きで表示。
- **見やすい色分け**: 風の強さに応じて段階的に表示を変更。

### 3. 台風情報の可視化
- **台風中心位置の表示**
- **予報円・暴風域・強風域の描画**
- **進路予測と過去経路の表示**
- **台風の詳細情報表示**: 中心気圧、最大風速、最大瞬間風速、強度などを確認可能。

### 4. 注意報・警報の可視化
- **全国の警報エリア表示**
- **注意報 / 警報 / 危険警報 / 特別警報の区分表示**
- **地域ごとの発表状況を一覧で確認可能**

### 5. 雨雲レーダー
- **気象庁の高解像度降水ナウキャストを表示**
- **時系列スライダーで過去〜最新コマを再生・移動可能**

### 6. 全国ランキング
- **気温・降水量・風速などのランキング表示**
- **気象状況を全国単位で素早く比較可能**

### 7. モダンなUI/UX
- **完全ブラウザ動作**: 追加のサーバー構築は不要。
- **レスポンシブ対応**: PC / タブレット / スマートフォンに対応。
- **ダークテーマ**: MapLibre を使ったシンプルで見やすい表示。
- **キャッシュ対応**: 地図データをブラウザキャッシュに保存して再利用。

## データソースと引用元 (Data Sources & Credits)

本プロジェクトは、以下の公開データおよびライブラリを利用して作成されています。  
データ提供元・開発者の皆さまに深く感謝申し上げます。

### 気象データ
- **気象庁 防災情報**
  - URL: https://www.jma.go.jp/
  - 用途: 気温、降水量、風向・風速、台風、警報・注意報、雨雲レーダーの表示。
- **気象庁 防災情報XML / GIS関連データ**
  - URL: https://www.data.jma.go.jp/developer/
  - 用途: 地図上での各種気象情報の可視化。

### 使用ライブラリ
- **MapLibre GL JS (v3.6.2)**: 地図レンダリング
- **Turf.js (v6.5.0)**: 空間演算・ポリゴン処理
